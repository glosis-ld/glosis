Abstract

This ontology is one of the modules comprising Global Soil Information System (GloSIS) ontology. The GloSIS ontology was developed part of the SIEUSOIL project, which aims at implementing and testing a shared China-EU Web Observatory platform that will provide Linked (Open) Data to monitor status and threats of soil and land resources. The GloSIS ontology enables the representation of soil related data in semantic format. The ontology has been derived from the UML GLOSIS data model v1.0, and it has been created in line with best practices and methodologies, reusing existing standard models and ontologies, including sosa/ssn for the representation of measurements, or skos for the representation of codelists.

Introduction back to ToC

The Global Soil Information System (GloSIS) ontology enables the representation of different soil properties, defined in the UN FAO Guidelines for soil description, the observed values and associated feature of interest. One of the main design principles for the implementation of this ontology was the alignment with existing standards, particularly those related to the representation of observations/measurements and associated properties like the SOSA/SSN ontologies. Additionally, it was important to represent the relations between the ontology elements and the related ISO standards elements (e.g., iso28258, iso19156), the connection with other geospatial vocabularies (e.g., GeoSparql ontology) and with vocabularies representing various quantity and unit standards (e.g., QUDT). Furthermore, the ontology represents multiple code lists that are associated to the soil properties. The GloSIS ontology was implemented following a modular approach, as a networked ontology, facilitating its reusability, extensibility and maintainability. This module is one of the 11 core modules in the GloSiS ontology:
    - glosis_cl
    - glosis_common
    - glosis_observation
    - glosis_surface
    - glosis_siteplot
    - glosis_profile
    - glosis_layer_horizon
    - glosis_procedure
    - glosis_unit
    - glosis_ext_property
     - iso28258
In particular this module contains soil properties that were not originally in the GloSIS domain model.
